## Прогнозирование уровня зарплаты по вакансиям

Модель машинного обучения для прогнозирования заработной платы по данным вакансий.

### Описание проекта

Проект реализует полный ML-pipeline:

* подготовка и очистка данных
* feature engineering (работа с JSON-полями, категориальными признаками, текстом)
* генерация эмбеддингов вакансий через SentenceTransformer
* обучение моделей регрессии
* сохранение артефактов
* подготовка предсказаний для новых данных

### Используемые технологии

* Python
* pandas / numpy
* scikit-learn
* XGBoost
* SentenceTransformers
* joblib


### Feature Engineering

В проекте реализованы:

* удаление нерелевантных и сильно разреженных признаков
* обработка JSON-колонок (`type`, `schedule`, `experience`)
* нормализация и категоризация городов
* добавление средней зарплаты по городу
* one-hot кодирование графика работы
* извлечение признаков из названия вакансии
* генерация текстовых эмбеддингов (384 признака)


### Модели

Обучены две модели:

* Linear Regression
* XGBRegressor

Целевая переменная логарифмируется (`log1p`) для стабилизации распределения.


### Структура проекта

```
├── dicts.py          # создание и сохранение вспомогательных словарей
├── features.py       # feature engineering
├── train.py          # обучение и сохранение моделей
├── prediction.py     # получение предсказаний
├── artifacts/        # сохранённые словари
├── models/           # обученные модели
└── submissions/      # файлы с предсказаниями
```
